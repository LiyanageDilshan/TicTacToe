# 🎮 Tic Tac Toe Android App

A simple and fun **Tic Tac Toe game** developed using **Android Studio (Java)**.
This project was created as a learning exercise to improve Android development skills while building a small interactive game.

## ✨ Features

🎲 **Dual Game Modes**

* **Play Solo** – Play against an intelligent computer opponent.
* **Play With a Friend** – Two players can play on the same device.

🔊 **Sound Effects**

* Move sound plays when a player or computer makes a move.

🧠 **Smart Computer Opponent**

* The computer uses a simple AI strategy to make smarter moves.

🎨 **Clean and Simple Interface**

* Easy-to-use layout designed for a smooth playing experience.

## 📱 Purpose of the Project

This app was created mainly to **improve programming and Android development knowledge**.
It demonstrates concepts such as:

* Activity navigation
* UI design with XML
* Game logic implementation
* Basic AI decision making
* Sound integration in Android apps

## 🛠 Technologies Used

* **Java**
* **Android Studio**
* **XML Layouts**
* **MediaPlayer (for sound effects)**

## 👨‍💻 Project Goal

The goal of this project is not to build a complex commercial game, but to create a **simple, functional, and educational Android application** that helps practice core development skills.
